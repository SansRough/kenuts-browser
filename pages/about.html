About This Project

KENUTS is a revolutionary browser built for the future. It uses a custom TCP-based protocol that offers:

  • Speed: Direct TCP communication without HTTP overhead
  • Simplicity: Minimalist design focused on content
  • Innovation: Custom protocol implementation
  • Education: Perfect for learning browser internals

Technical Features

Custom Protocol

KENUTS protocol runs over raw TCP sockets, providing direct communication between client and server.

Electron Framework

Built with Electron for cross-platform compatibility and modern web technologies.

Go Backend

Powered by a lightweight Go server that handles file serving and protocol implementation.

Developer Friendly

Keyboard shortcuts, tabbed interface, and developer tools integration.
